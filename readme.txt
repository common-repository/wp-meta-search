Search form for searching through custom fields, post types, categories, tags, and free text.

== Description ==

This is a flexible search form widget.
You can easily generate a multi condition search form and display anywhere as widget.

複数のカスタムフィールド、投稿タイプ、カテゴリー、タグでの検索が可能な複合検索フォームを簡単作成できるプラグイン。ウィジェットで好きな場所に設置可能。

Currently supports
- Custom Fields
- Taxonomies
- Categories
- Post tags
- Post Type


== Installation ==



1. Upload the plugin files to the `/wp-content/plugins/wp-meta-search` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->WP Meta Search screen to configure the plugin
4. Use the Appearance->Widgets and register 'WP Meta Search Form' to display the form as a widget
